# VoltPure Home Assistant Add-ons

Officiële HA-add-on repository voor VoltPure EMS klanten.

## Add toe aan HA

1. **Settings → Add-ons → Add-on Store**
2. Menu (⋮) → **Repositories**
3. Voeg toe: `https://github.com/zwijn25/voltpure-ha-addons`
4. Refresh → installeer **VoltPure Edge**

## Beschikbare add-ons

- **vp-edge** — Edge data-publisher voor cloud (license-gated)
